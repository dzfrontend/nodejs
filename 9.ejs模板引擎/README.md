# ejs模板引擎基础知识  

express模板引擎：jade模板引擎和ejs模板引擎

jade模板引擎: 破坏式，侵入式模板引擎，和普通的html,css不能共存

ejs模板引擎：非侵入式，不会破坏原来的html,css


## 安装

###
1.肯定要有node.js环境啦

2.然后
```
npm install ejs -g
```
全局安装  

3.安装到本地
```
npm install ejs --save
```
